# 📖 다락방 성경통독 앱

교회 소모임(다락방) 순장과 순원들이 **함께 성경통독**을 진행하고 진도를 관리하는 모바일 퍼스트 웹앱입니다. 스마트폰 브라우저에서 사용하며, 모든 데이터는 기기에 오프라인으로 저장됩니다.

## ✨ 주요 기능

- **순원 관리** — 순원을 수시로 추가/편집/삭제
- **개인별 통독 계획** — 달력에서 시작일·종료일을 고르면 전체 1,189장을 기간으로 나눠 **하루 읽을 분량을 자동 계산**
- **시작 위치 선택**
  - 구약부터: 창세기 → 말라기 → 마태복음 → 요한계시록
  - 신약부터: 마태복음 → 요한계시록 → 창세기 → 말라기
- **오늘의 읽기 안내** — 매일 어디부터 어디까지 읽어야 할지 표시
- **장별 라디오버튼 그리드** — 66권 전 장을 동그라미 숫자 버튼으로 표시
  - 🔴 오늘 읽을 장 (깜빡임) · 🔵 읽은 장 · 🟠 밀린 장 · ⚪ 예정 장
- **순원별 진도 비교 차트** — 계획 대비 달성률을 막대그래프로 비교
- **달성률 레벨 배지** — 🌱 씨앗부터 😇 완주까지 9단계 격려 아이콘

## 🛠 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| 프레임워크 | React 18 + Vite 5 |
| 스타일 | Tailwind CSS 3 |
| 상태관리 | React `useState` + `useContext` + `localStorage` |
| 라우팅 | React Router 6 (하단 탭 네비게이션) |
| 차트 | Recharts 2 |

```bash

npm run dev      # 개발 서버 (http://localhost:5173)

```

## 🧮 통독 계산 로직

1. **하루 분량** = `Math.ceil(1189 / (종료일 − 시작일 + 1))`
2. **통독 순서 배열**을 시작 위치(구약/신약)에 따라 1,189개 `{책, 장}`으로 평탄화
3. **오늘 범위** = `(오늘 − 시작일)`번째 날 기준으로 누적 인덱스를 잘라 산출
4. **장 상태** = 읽음 / 오늘 / 밀림(지난 분량 미독) / 예정 으로 구분

## 📱 사용 흐름

1. **홈**에서 순원을 추가합니다.
2. 순원을 선택하면 **내 통독** 화면으로 이동합니다.
3. **통독 계획 세우기**에서 기간과 시작 위치를 정합니다.
4. 매일 **오늘의 읽기** 분량을 확인하고, 읽은 장을 탭하거나 "오늘 분량 다 읽었어요"로 체크합니다.
5. **진도** 탭에서 순원들과 달성률을 비교하며 서로 격려합니다.

## 💾 데이터 저장

- 저장 키: `localStorage["darakbang_app"]`
- 모든 데이터는 **이 기기에만** 저장됩니다. 설정 화면의 **백업 내보내기/가져오기**로 기기 간 이전이 가능합니다.
